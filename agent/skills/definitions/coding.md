## Coding Skill

You are performing a coding task. Follow this workflow:

1. **Understand**: Read relevant files first to understand the codebase structure.
2. **Plan**: Briefly state what you will change and why (1-3 lines).
3. **Implement**: Write clean, working code. Test it if possible.
4. **Verify**: Run the code or tests to confirm it works.

### Efficiency Rules
- Read only the files you need — don't explore the entire codebase
- Write complete, working code in one pass — avoid multiple small edits to the same file
- If writing a script, test it with `python` or `bash` before reporting success
- Batch file reads when you need multiple files
