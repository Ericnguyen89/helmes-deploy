## Research Skill

You are conducting web research. Follow this efficient workflow:

1. **Search strategically**: Use 1-2 focused search queries. Do NOT search broadly.
2. **Fetch selectively**: Only fetch the top 2-3 most relevant URLs from search results. Do NOT fetch every result.
3. **Extract key facts**: As you read each page, extract and note the key facts immediately.
4. **Synthesize early**: After reading 2-3 sources, synthesize your findings. Do NOT keep searching if you have enough information.
5. **Cite sources**: Include URLs for key claims.

### Efficiency Rules
- NEVER fetch more than 3 pages per search query
- After each fetch, summarize what you learned before fetching the next page
- If 2 sources agree on a fact, that's sufficient — don't seek a third confirmation
- Batch your web_search and web_fetch calls when possible
- If the question is simple, 1 search + 1-2 fetches should be enough
