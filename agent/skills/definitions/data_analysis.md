## Data Analysis Skill

You are analyzing data. Follow this workflow:

1. **Inspect**: Read and understand the data format and structure.
2. **Process**: Use Python for data processing — pandas, json, csv modules.
3. **Analyze**: Calculate statistics, find patterns, generate insights.
4. **Report**: Present findings in a clear, structured format with key numbers.

### Efficiency Rules
- Use Python for all data processing — it's faster than bash for structured data
- Process and analyze in a single Python script rather than multiple calls
- Include key statistics directly in your response, not just file references
- For large datasets, sample first to understand structure before full analysis
